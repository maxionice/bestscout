# Feature parity specification

This document defines “complete” for BestScout 1.0. A checkbox requires an
automated test where practical and a manual acceptance record on a supported FM26
build. Read and write capabilities are tracked separately.

## Scouting and analysis

- [ ] Global player, staff, club and competition search
- [ ] Current and potential ability, reputation and hidden attributes
- [ ] Advanced filters, custom columns and saved views
- [ ] Ratings for every supported position and FM26 in/out-of-possession role
- [ ] Peak-attribute projection and development probability
- [ ] Wonderkid, bargain, free-agent and expiring-contract lists
- [ ] Player comparison, similar-player search and replacement suggestions
- [ ] Squad depth, age, wage, contract and succession analysis
- [ ] Shortlists, favourites, tags, notes and CSV/JSON/HTML export

## Player and staff editing

- [ ] Identity, appearance, nationality, languages and relationships
- [ ] Technical, mental, physical, goalkeeping and hidden attributes
- [ ] CA, PA, reputation, positions, preferred foot and preferred moves
- [ ] Contract, wages, bonuses, clauses and squad registration
- [ ] Fitness, condition, morale, happiness, injuries and bans
- [ ] Staff roles, responsibilities and coaching/scouting attributes

## Clubs, transfers and competitions

- [ ] Club finance, debts, facilities, reputation, attendance and stadium
- [ ] Kits/colours and club relationships
- [ ] Immediate moves, swaps and future transfers
- [ ] Competition reputation, standings, points and supported fixtures/stages

## Automation and safety

- [ ] Preset manager and filtered mass edit with preview
- [ ] Attribute freezer with per-field increase policy
- [ ] Automatic save backup, write verification, journal and undo
- [x] Build fingerprinting and read-only fallback for unknown versions
- [ ] Newgen/facepack assignment tools
